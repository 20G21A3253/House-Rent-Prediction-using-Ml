Other

Here we have analyzed other aspects of the houses in each city which are

  1. Seller Type - The entity which is putting the property on rent - Can be of 3 types
    • Agent - The real estate agent through which the owner would like to put the property on rent
    • Builder - The builder of the property who might put the property on rent
    • Owner - The owner may directly put the property on rent
  2. Layout Type - The layout of the property put on rent
    • RK - The property contains only a Room and Kitchen where the bedroom and hall are combined
    • BHK - The property contains a Bedroom, Hall and a Kitchen
  3. Property Type - The type of property which is put on rent
    • Apartment - A flat in a building which contains many flats
    • Studio Aparment - Contains only one room
    • Independent House - A single house with boundary walls on four sides
    • Independent Floor - An entire floor on a building isolated from the rest of the building
    • Villa
    • Penthouse
  4. Furnish Type
    • Semi Furnished - Contains some furniture sufficient enough for the tennant to settle
    • Furnished - Contains all the necessary furniture for a tennant to settle
    • Unfurnished - Contains no furnitire
